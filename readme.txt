Fixed Social Icons for your wordpress website

== Description ==
Wp Fixed Social profile Icons  places the social icons on the left/right side of the page.

When a user rollover the icon the full icon is shown with nice animation

You can send your feedback from my <a href="http://amansaini.me">website</a>

== Installation ==
Download and unzip the plugin

Upload the entire fixed-social-profile-icons/ directory to the /wp-content/plugins/ directory

Activate the plugin through the Plugins menu in WordPress

Enable the icons you want to show on your site.

Add your profile links

Save it and check in frontend !
